# NativEats
## Getting Started
```
npm install
// Starts API server
npm run dev -w api
// Starts RN app on iOS simulator
npm run ios -w app
// Use if above ios command does not work
npm run web -w app
// To install a package to either the app or api directory
npm install PACKAGE -w DIRECTORY
// To install a package as a dev dependency to either the app or api directory
npm install PACKAGE -D -w DIRECTORY
